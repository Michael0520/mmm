# MMM Christmas

A pixel-art Christmas interactive web app.

**MMM** = **M**ichael + **M**eiko + **M**erry Christmas

## Tech Stack

- **Framework:** TanStack Start
- **UI:** 8bitcn/ui (pixel-style components)
- **Icons:** pixelarticons + custom pixel art
- **Styling:** Tailwind CSS v4
- **Font:** Press Start 2P

## Pages

| # | Route | Description |
|---|-------|-------------|
| 1 | `/` | Landing - MMM Christmas welcome |
| 2 | `/rps` | Rock Paper Scissors game |
| 3 | `/vinyl` | Vinyl player + Christmas music |
| 4 | `/cat` | Cat display (dining time) |
| 5 | `/photo` | Photo frame |
| 6 | `/ending` | Merry Christmas message |

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
